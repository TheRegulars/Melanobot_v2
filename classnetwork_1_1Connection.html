Detailed Description

Abstract service connection.

Instances of this class will be created by Melanobot from the configuration and will send Message objects and receive either Command or OutputMessage objects.

To create your own class, inherith this one and override the pure virtual methods. Then use REGISTER_CONNECTION() to register the class to the ConnectionFactory.

It is recommended that you add a static method called create in the derived class with the following signature:

YourClass* create(Melanobot* bot, const Settings& settings);

to be used with REGISTER_CONNECTION.

You should also have a corresponding log type to use for the connection, registered with REGISTER_LOG_TYPE().
