Detailed Description

Message handler abstract base class.

To create concrete handlers, inherit this class and specialize the virtual methods (see their own documentation).

For simple cases, inherit SimpleAction and specialize on_handle().

To make a handler visible to HandlerFactory, use Melanomodule::register_handler.

The constructor of a handler class must have the same signature as the one used here to work properly.
