Detailed Description

template<class CharT, class Traits = std::char_traits<CharT>>
class string::BasicQuickStream< CharT, Traits >

Quick and simple unformatted string input stream.

Invariant
  • pos Points to the next character returned by next()
  • if eof() returns true, next() returns Traits::eof()
Note
Part of the interfacte is different from std::istream as it behaves differently from it
